I - Accès au site
	lancez index.php pour avoir accès au site Pool d'eau, si vous utilisez localhost 
	avec Wamp le site se lancera tout seul en ouvrant le dossier.

II - BDD
	Chargez la bdd avec le fichier bddpiscine.sql, le fichier marche avec Phpmyadmin
	pour la remplir vous pourrez utilisez le bouton présent sur la page d'acceuil avec
	un fichier .csv qui respecte le format défini dans le cahier des charges. Vous pouvez 
	par exemple utiliser 2013-2017-50.csv présent dans le dossier
	----------IMPORTANT --------
		les paramètre de connexion pour la bdd sont réglés sur nom = root, pas de mdp
